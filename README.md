An award-worthy, immersive 3D cinematic journey through the backend infrastructure of a Senior Backend Engineer. This portfolio replaces standard scrolling lists with a high-fidelity WebGL traversal through the core components of modern distributed systems.

## 🚀 The Journey

Traverse 8 distinct system checkpoints, each representing a core architectural layer:

1.  **Sys-Boot**: Hacker-style terminal logon and kernel initialization.
2.  **The Gateway**: Initial wireframe traversal through the ingress layer.
3.  **Kafka Highway**: High-throughput message streaming with 1,500+ real-time particles.
4.  **Microservices City**: Orchestrated service nodes with elastic autoscaling visualizations.
5.  **Database Vault**: A monolithic hexagonal core protected by rotating network security rings.
6.  **K8s Cloud**: Floating crystal-pod abstraction layer for container orchestration.
7.  **Open Source Orbit**: A solar-system scale visualization of contributions to Apache Fineract and GSoC.
8.  **Response Terminal**: Secure connection establishment for final contact.

## 🛠 Technical Stack

Built with a performance-first approach to 3D rendering in the browser:

*   **Core**: [Next.js 14](https://nextjs.org/) (App Router, Server Components)
*   **3D Engine**: [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) & [Three.js](https://threejs.org/)
*   **State Management**: [Zustand](https://github.com/pmndrs/zustand) (Persistent system store)
*   **Animations**: [GSAP](https://greensock.com/gsap/) (Cinematic timing) & [Framer Motion](https://www.framer.com/motion/) (UI/Overlays)
*   **Effects**: [PostProcessing](https://pmndrs.github.io/postprocessing/public/docs/) (Bloom, Chromatic Aberration, Noise)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Smooth Scroll**: [Lenis](https://lenis.darkroom.engineering/)

## 📦 Getting Started

### Prerequisites

*   Node.js 18+
*   npm / pnpm / yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/Dpk376/storymodeportfolio.git
    cd storymodeportfolio
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
